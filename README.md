# LittleComputerPeople

Remake of the 80s classic, built in Unity.

![Alt text](http://i.imgur.com/31sOZjW.png "Screenshot")

Keypress commands:

v = watch tv

s = go to sleep

t = use toilet

b = read book

w = write (typewriter)

e = exercise

f = make food

a = alarm clock (wake up)

c = call him

esc = quit
